# Chilaquiles Theme Palette

## Design Principles

- **Dark variant**: Light shades of accent hues on near-black background. All token colors pass WCAG AA (4.5:1 minimum).
- **Light variant**: Dark shades of accent hues on warm cream background. All token colors pass WCAG AA.
- Hue identity is preserved across variants (green functions stay green, red keywords stay red).
- Inspired by: Dracula (vivid saturation on dark), Monokai (bright function green), Night Owl (warm low-chroma backgrounds).
