# Chilaquiles Theme

A VS Code color theme inspired by the Mexican dish chilaquiles — warm chili reds, toasted corn golds, and cilantro greens on a charred dark background (or crisp crema white for light mode).

## Variants

- **Chilaquiles Dark** — Night Comal background (`#1A1008`) with vibrant accent colors
- **Chilaquiles Light** — Queso Fresco background (`#FDF8F0`) with contrast-adjusted tones

## The Palette

| Role      | Dark                          | Light                        |
| --------- | ----------------------------- | ---------------------------- |
| Keywords  | Vivid Tomato `#E05C4B`        | Deep Chili `#8B2318`         |
| Strings   | Bright Corn Gold `#E8B84B`    | Dark Gold `#6B5000`          |
| Functions | Bright Cilantro `#6DBF72`     | Deep Cilantro `#145218`      |
| Types     | Warm Chili `#D45C4C`          | Deeper Red `#7A1E10`         |
| Comments  | Lightened Adobe `#A67B6A`     | Deep Adobe `#6B4237`         |

All token colors meet WCAG AA contrast requirements (4.5:1 minimum) on their respective backgrounds.

## Features

- **Full editor UI theming** — activity bar, sidebar, tabs, status bar, terminal, notifications, inputs, and more
- **Semantic token support** — enhanced highlighting for TypeScript/JavaScript via language server
- **Language-specific overrides** — TS/JS generics, arrow functions, interface names, built-in types

## Installation

Install via the VS Code Marketplace: search for **Chilaquiles Theme**.

Then: `Ctrl+K Ctrl+T` (or `Cmd+K Cmd+T` on Mac) → select **Chilaquiles Dark** or **Chilaquiles Light**.
